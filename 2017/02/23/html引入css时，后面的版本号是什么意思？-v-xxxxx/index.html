html引入css时，后面的版本号是什么意思？(?v=xxxxx)

发表于 2017-02-23

转自：http://chasen-g.cn/
在查看一些网站的源代码时，我们会遇到CSS引入后面加了个类似版本号的代码，这是什么意思呢？

举个例子，我们在做开发工作的时候，有时候会遇到这种情况：修改完网页的样式，但是保存刷新后，预览效果却没有变化。如果是新手在遇到这个问题的时候，一般会用换浏览器，或者Ctrl+F5 （强制刷新）的方法来解决。

而版本号的加入就是为了可以让浏览器自动完成这个工作，让修改效果和预览效果能够同步。

因为网站部署在服务器上，用户浏览网页时也会出现这个情况，如果CSS引入的代码时这样的：

用户第一次浏览这个网页的时候，浏览器从服务器上下载base.css这个样式文件，存储到缓存中。

这时，开发人员在远程修改了这个样式，但是样式文件名字没有变，用户在浏览的时候，有可能就会看不到修改后的样式，这就是修改和浏览没同步。
