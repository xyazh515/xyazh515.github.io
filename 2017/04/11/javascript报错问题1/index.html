今天写代码遇到过一个问题，那就是把一段js代码放在标签中，出现了Uncaught TypeError: Cannot set property ‘onclick’ of null的提示。后来去网上搜了下，大概明白了原因。

而节点需要先加载完才能执行onclick事件，所以，先让节点加载完之后，再执行JS。

或者，把JS写在里面，这种写法，浏览器是先加载完节点，才开始执行JS。
